# 支持的中间件列表

## 数据库

### MySQL / MariaDB
- **插件名称**: `mysql`
- **版本**: v1.0.0
- **诊断能力**:
  - 慢查询检测
  - 死锁分析
  - 连接池监控
  - 索引使用分析
  - 主从同步延迟检测

### PostgreSQL
- **插件名称**: `postgresql`
- **版本**: v1.0.0
- **诊断能力**:
  - 查询性能分析
  - 锁等待检测
  - 表膨胀检测
  - 复制延迟监控

### Memcached
